ReBOOT

Project Purpose

- ReBOOT is a supportive web application designed to help individuals stay motivated, mentally resilient, financially aware, and career-ready during periods of unemployment.
- Instead of treating unemployment as downtime, Reboot reframes it as a productive reboot phaseâ€”a time for growth, learning, and preparation.

Project Scope

- Currently focused on IT and tech-related careers

- Future versions will expand to support other career paths

Project Core Idea

Unemployment can impact motivation, confidence, and direction. Reboot provides a structured, supportive environment that helps users:

- Stay mentally engaged and positive

- Track progress and routines

- Prepare for job applications and skill growth

- Manage finances during job gaps
